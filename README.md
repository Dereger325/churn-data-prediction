# Customer Churn Prediction

Predicting which telecom customers will leave using machine learning.

## Dataset
- Source: IBM Telco Customer Churn
- Samples: 7,043 customers
- Features: 21
- Target: Churn (Yes/No)
- Churn rate: ~27%
